## 📌 Project Overview

**AI Superstore Predictive Analytics v3** is an end-to-end data analytics and machine learning dashboard designed to transform raw retail data into actionable business insights. This project demonstrates how modern analytics tools and AI models can help businesses understand customer behavior, forecast sales, optimize discounts, and improve decision-making.

The dashboard provides interactive visualizations, predictive modeling, customer segmentation, and automated insights in a clean, professional interface suitable for business intelligence and data science portfolios.

### 📊 Trends Dashboard

* Sales, Profit, and Quantity analysis
* Year-over-Year (YoY) performance tracking
* Monthly and category-wise trends
* Interactive charts and filters

* Region and state-level sales visualization

* Top-performing products
* Top customers and regions
* Sales ranking system

### 🧩 Discount Analyzer

* Discount impact on profit analysis
* Profit vs Discount visualization
* Optimization insights for pricing strategies

### 🤖 AI-Powered Gemini Q&A

* Ask questions about data using AI
* Natural language insights
* Automated analytics responses

### 📤 Export Functionality

* Export reports and filtered data
* Download analytics results

## ⚙️ Installation & Setup

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/ai-superstore-analytics-v3.git
cd ai-superstore-analytics-v3
```

### Step 2: Create Virtual Environment

```bash
python -m venv venv
```

### Step 3: Activate Environment

**Windows:**

```bash
venv\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

### Step 4: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 5: Run the Application

```bash
streamlit run app/dashboard_ai_prov2.py
```
